# 📊 AI Smart Data Dashboard

A **colorful and interactive data visualization dashboard** built with **Streamlit, Pandas, and Plotly**.
This application allows users to **upload CSV or Excel files and instantly generate insights, charts, and analytics**.

---

# 🚀 Features

✔ Upload **CSV / Excel datasets**
✔ Automatic **dataset preview**
✔ **Dataset statistics** (Rows, Columns, Missing values, Duplicates)
✔ **Interactive filters** for categorical columns
✔ **KPI metrics** (Mean, Max, Min, Sum)
✔ Multiple **interactive visualizations**

### 📊 Charts Included

* Histogram
* Box Plot
* Line Chart
* Scatter Plot
* Area Chart
* Violin Plot
* Bar Chart
* Pie / Donut Chart
* Treemap
* Correlation Heatmap
* Custom Scatter Plot (User selects X and Y columns)

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/ai-smart-dashboard.git
cd ai-smart-dashboard
```

Install required libraries:

```bash
pip install streamlit pandas plotly openpyxl
```

---

# ▶️ Run the Application

Start the Streamlit app:

```bash
streamlit run app.py
```

Then open the browser:

```
http://localhost:8501
```
